========================================================================
WSR AGENT INTEGRATION & SYNCHRONIZATION GUIDE
========================================================================
Bản hướng dẫn này giúp tích hợp bộ quy tắc WSR (GEMINI.md, Rules, Workflows)
vào các IDE và trợ lý AI khác nhau (Claude, Codex, Cursor, Antigravity).

------------------------------------------------------------------------
1. CURSOR IDE (.cursorrules)
------------------------------------------------------------------------
Để Cursor Agent tự động nạp các quy tắc của dự án:
- Tạo một liên kết mềm (Symbolic Link) hoặc sao chép file GEMINI.md
  từ thư mục dự án ra ngoài thư mục gốc (root) và đổi tên thành:
  .cursorrules
  Lệnh tạo Symlink trên Windows (Powershell):
  cmd /c mklink .cursorrules .agents\global\GEMINI.md
- Hoặc copy trực tiếp file GEMINI.md vào file .cursorrules ở root dự án.

------------------------------------------------------------------------
2. CLAUDE DEV / ROO CODE / CLAUDE IN VSCODE (.clauderules)
------------------------------------------------------------------------
Để các extension Claude tự động tuân thủ các quy chuẩn WSR:
- Sao chép nội dung của GEMINI.md hoặc tạo file .clauderules tại
  thư mục gốc dự án để extension tự động đọc khi bắt đầu phiên làm việc.
  Lệnh tạo Symlink trên Windows (Powershell):
  cmd /c mklink .clauderules .agents\global\GEMINI.md
- Hoặc khai báo đường dẫn trỏ tới .agents/global/GEMINI.md trong phần
  "System Instructions" của cấu hình custom agent.

------------------------------------------------------------------------
3. CODEX-NATIVE (.agents/AGENTS.md)
------------------------------------------------------------------------
Để môi trường Codex nhận diện và định vị chính xác tri thức:
- Tạo file AGENTS.md tại thư mục gốc của dự án.
- Khai báo các đường dẫn Canonical trỏ tới các file quy tắc, kỹ năng
  và playbook cục bộ của dự án dưới dạng liên kết markdown.
- Codex sẽ tự động đọc AGENTS.md khi mở dự án để định hướng hành vi.

------------------------------------------------------------------------
4. ANTIGRAVITY IDE (Windows AppData Sync & /reload)
------------------------------------------------------------------------
Để nạp và đồng bộ cấu hình trong IDE Antigravity:
- Đảm bảo thư mục .agents được cấu hình ở root dự án.
- Chạy script đồng bộ hóa để cập nhật cấu hình toàn cục lên Windows AppData:
  python .agents/scripts/sync_config.py
- Trong cửa sổ chat của IDE, gõ lệnh `/reload` để nạp và ghi nhớ
  các rules và workflows mới nhất vào bộ nhớ của Agent.
