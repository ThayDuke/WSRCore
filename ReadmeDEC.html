Các từ viết tắt hệ thống

Các thuật ngữ và công cụ cốt lõi được định nghĩa thống nhất trong toàn bộ tài liệu hướng dẫn và mã nguồn.

TL / TQ / TS

Tương ứng với: ToolListening (Nghe), ToolQuiz (Trắc nghiệm), ToolSlide (Bài giảng).

TE / TA / TQI

Tương ứng với: ToolExam (Kiểm tra), ToolAdaptive (Thích ứng), ToolQbankImport (Nhập ngân hàng câu hỏi).

GL / GC / GB

Tương ứng với các phân vùng Game: Game Lobby (màn hình trung gian), Game Castle (Lâu đài), Game Battle (Trận đấu).

DEC Master UI (DMU 2026)

Nghiêm cấm tuyệt đối việc hardcode màu sắc, spacing, typography hoặc border. Mọi phần tử giao diện mới phải kế thừa token hệ thống.

Sử dụng Token var()

Sử dụng các token chuẩn hóa var(--dec-*) và lớp CSS dec-ui-* để nhất quán kiểu dáng.

Icon-free Tiêu Đề

Tuyệt đối không dùng icon trong H1/H2, nút bấm hoặc tab nhãn. Chỉ dùng khi có yêu cầu đánh dấu bằng [DMU-EXCEPTION].

Miễn trừ cho ToolGame

ToolGame (GL, GC, GB) được miễn trừ hoàn toàn DMU để tùy biến giao diện game tự do sáng tạo.

Đồng bộ hóa Theme Youth & Elegant

Mọi thay đổi giao diện làm xê dịch layout bắt buộc phải cập nhật đồng thời cho cả giao diện Youth (mặc định) và Elegant.

Theme Youth

Theme tươi sáng, trẻ trung sử dụng màu sắc gradient sống động, bo góc mềm mại.

Theme Elegant

Theme tối giản, sang trọng sử dụng đường nét sắc sảo, bảng màu dịu nhẹ tinh tế.

Triệt tiêu Jitter

Cập nhật padding, margin đồng bộ giúp tránh hiện tượng nhảy layout khi chuyển đổi qua lại giữa hai theme.

Điểm neo hồi quy kỹ thuật (Anchors)

Các cơ chế kỹ thuật ngầm bắt buộc phải giữ nguyên để duy trì tính hoạt động của hệ thống local.

Cổng Local 6868

Khởi chạy localhost luôn sử dụng cổng kết nối cố định 6868 cho toàn dự án.

Ẩn Console Windows

Khi chạy local server, console Windows bắt buộc phải được ẩn bằng thư viện ctypes.

Tách âm TTS

Logic chia nhỏ văn bản sang TTS phải tự động loại bỏ tiền tố tên người nói (ví dụ: "A:", "B:").

Sao lưu & Chạy wsr_audit.py Cục Bộ

Cách thức sao lưu dữ liệu an toàn và cơ chế chấm điểm tĩnh được tích hợp tại DEC.

Backup dec* Shortcut

Phím tắt sao lưu nhanh dự án nén thành file zip, lưu trực tiếp vào ổ đĩa E:\BACKUP DukeEnglishCenter.

Kiểm tra đồng bộ Theme

Trình audit cục bộ wsr_audit.py quét và trừ 15 điểm nếu phát hiện sửa đổi CSS một theme mà bỏ quên theme kia.

Chống lỗi Mojibake

Quét mã hóa chặt chẽ ngăn chặn moijbake tiếng Việt trong các template HTML và bài giảng Ultimate.
