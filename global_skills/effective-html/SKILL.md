# KỸ NĂNG TẠO ARTIFACT HTML TRỰC QUAN (effective-html)

---

## 1. htmls
*   **Mục tiêu:** Tạo file HTML tự chứa (self-contained) mô tả thông tin.
*   **Áp dụng:** Khi cần báo cáo, explainer, so sánh, slide, prototype dạng HTML.
*   **Yêu cầu:** Thiết kế gọn gàng, trực quan, dễ tương tác.

---

## 2. htmld
*   **Mục tiêu:** Vẽ sơ đồ kiến trúc hệ thống, stack bằng HTML/SVG chất lượng cao.
*   **Áp dụng:** Khi cần trực quan hóa nhanh cấu trúc hệ thống, không dùng nhiều văn bản.
*   **Yêu cầu:** Sử dụng SVG chất lượng cao, tối ưu hiển thị toàn màn hình.

---

## 3. htmlp
*   **Mục tiêu:** Tạo trang kế hoạch hành động thực tế, đơn giản, tổ chức trực quan.
*   **Áp dụng:** Khi cần đề xuất các bước triển khai chi tiết cho người dùng phê duyệt.
*   **Yêu cầu:** Câu từ ngắn gọn, bám sát yêu cầu, làm sạch ngữ pháp.
